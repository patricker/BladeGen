User Guide

Basics
- Use the left sidebar to adjust Blade, Guard, Handle, and Pommel.
- Hover a section header to highlight that part in the 3D view.
- Shuffle buttons randomize only that category; Randomize buttons at top affect all.

Key Controls
- Blade: Length/width/thickness set dimensions; Curvature bends the blade.
- Chaos adds subtle edge roughness; Asymmetry widens left/right edges differently.
- Fullers: enable to show decorative grooves; Depth/Length tune their look.
- Guard: Width/Thickness plus Curve/Tilt; Style presets: bar/winged/claw.
- Handle: Ridges toggle; Wrap (geometry) and Wrap Texture (visual stripes).
- Other: Blade Detail controls mesh smoothness; Stylization exaggerates proportions.

Exports
- GLB: export to Blender/engines.
- OBJ/STL: alternate 3D formats.
- SVG: 2D blade outline for blueprints.

Tips
- Use Blade Detail 64–160 for smooth curves; lower for performance.
- If fullers look too short on small blades, reduce Fuller Length or disable.
- Use Stylization sparingly for dramatic fantasy looks.
