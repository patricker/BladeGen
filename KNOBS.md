Knob Ideas:

🔪 Blade (15+ knobs)

Blade length (overall length).

Blade base width (width at guard).

Blade tip width (width at point).

Blade thickness (spine thickness).

Taper ratio (how quickly it narrows from base to tip).

Curvature (straight ↔ curved arc).

Twist angle (spiral/twisted blades).

Fuller depth (groove indentation).

Fuller length (percentage of blade).

Number of fullers (1–3 grooves).

Serration frequency (number of teeth).

Serration amplitude (tooth depth).

Tip shape (slider between pointed, rounded, leaf-shaped).

Cross-section profile (diamond ↔ lenticular ↔ hexagonal).

Edge bevel angle (sharp vs thick edge).

Symmetry toggle (double-edged vs single-edged).

Noise/chaos amount (jagged edges, fantasy distortions).

🛡 Guard / Crosspiece (10 knobs)

Guard width (span across blade).

Guard thickness (front-to-back depth).

Guard height (up-down from blade).

Guard curvature (straight ↔ arched).

Guard tilt (angled toward blade or handle).

Number of quillons (arms/branches).

Quillon length (if >2).

Guard ornamentation complexity (flat ↔ ornate).

Guard tip style (blunt ↔ spiked ↔ clawed).

Symmetry toggle (symmetric vs asymmetric guard arms).

✋ Handle / Grip (8 knobs)

Handle length (short dagger grip ↔ greatsword two-hander).

Handle radius (top).

Handle radius (bottom) (enables taper).

Grip segmentation count (rings, bands).

Grip flare at pommel (outward flare toggle).

Handle curvature (straight ↔ slightly bent).

Wrap pattern complexity (smooth ↔ ridged ↔ spiral).

Tang visibility (hidden ↔ full tang).

⚖️ Pommel (7 knobs)

Pommel size (relative scale).

Pommel shape morph (sphere ↔ disk ↔ spike).

Pommel elongation (flat ↔ long).

Pommel offset (centered ↔ angled).

Pommel facet count (smooth ↔ gem-like).

Balance weight ratio (relative to blade mass).

Pommel spike length (if elongated type selected).
